공학수학2 - 챕터 13.1-13.2

Chapter 13. Complex Numbers and Functions, complex differentiation

  1. 복소수

    1. 개요
      1. 방정식에서 실수가 아닌 근
      2. 전기공학자들은 전류기호 $i$와 혼동을 피하고자 $j$를 쓰기도 한다.
      3. MISC
        1. 복소수를 처음 사용한 사람은 이탈리아 수학자 Girolamo Cardano이고, 복소수라는 용어는 Carl Friedrich Gauss가 처음으로 도입했다.
    2. 표기
      1. $z(x,y)-x+iy$로 표기
        1. $x=Rez$ : 실수부
        2. $y=Imz$ : 허수부
      2. $i=(0,1)$
        1. 단위허수imaginary unit라고 부른다.
      3. $x=0$일 때, $z=iy$
        1. 순허수pure imaginary라고 부른다.
    3. 계산
      1. 나눗셈
        1. 분모에 켤레복소수를 곱한다.
        2. $\frac{8+3i}{9-2i}=\frac{(8+3i)(9+2i)}{(9-2i)(9+2i)}=\frac{66+43i}{85}$
    4. 복소평면
      1. 복소수를 기하학적으로 표현해본다.
      2. 표면
        1. x축은 실축real axis, y축은 허축Imaginary axis라 한다.
        2. 이때 실축과 허축으로 이루어진 좌표계를 직교좌표계라 한다.
        3. 복소수가 실축과 허축에 따라 표시되는 xy 평면을 복소평면 이라 부른다.
      3. MISC
        1. 프랑스 수학자 Jean Robert Argano의 이름을 따라 Argand 다이어그램이라고도 부른다.
      4. 공액복소수Complex conjuage number
        1. 정의
          1. 복소평면에서 실수축에 대칭인 복소수
          2. $\bar{z}=x-yi$
        2. 활용
          1. $Rez=x=\frac{1}{2}(z+\bar{z})$
          2. $Imz=y=\frac{1}{2}(z-\bar{z})$
            1. 주의 : $Imz$에는 $i$가 붙지 않는다.
              $z=1+2i$일 때 $Imz$는 2다. $2i$가 아니다.

  2. Polar form of complex numbers

    1. 표기

      1. xy 좌표 대신 $r, \theta$로 표기한다.
      2. $x=rcos\theta, y=rsin\theta$
      3. r
        1. z의 절댓값(absolute value 또는 modulus)
        2. $\left|z\right| $로 표기
        3. $r=\left|z\right|=\sqrt{x^2+y^2}=\sqrt{z\bar{z}}$
      4. $\theta$
        1. z의 편각argument
        2. $argz$로 표기
        3. $\theta=argz=tan^{-1}\frac{y}{x}$

    2. $-\pi<Argz<\pi$

      1. $Argz$는 $argz$의 주값principal value
      2. 왜 이런 값을 두냐면, 이런 정의가 없다면 복소수의 polar form 값이 너무 다양해지기 때문이다.
        1. $2cos\pi+i2sin\pi=2cos3\pi+i2sin3\pi$ … 등등.
      3. $argz=Argz\pm2n\pi$ (n은 정수)

    3. 교수님 : Argz의 표현

      1. $arctan$의 값은 아래와 같이 정의된다.

        if Argz
        x>0, y>0 $tan^{-1}\frac{y}{x}$
        x>0, y<0 $tan^{-1}\frac{y}{x}$
        x<0, y>0 $\pi + tan^{-1}\frac{y}{x}$
        x<0, y<0 $-\pi + tan^{-1}\frac{y}{x}$

      2. 진짜 유용하게 쓰인다.

        1. $z=1+i$ : $arctan{\frac{1}{1}}=\frac{\pi}{4}=Argz$
        2. $z=-1-i$ : $arctan{\frac{-1}{-1}}=\frac{\pi}{4}$$,
          Argz=-\pi+\frac{\pi}{4}=-\frac{3\pi}{4}$
        3. $z=-2+2i$ : $arctan{\frac{2}{-2}}=-\frac{\pi}{4}$$,
          Argz=\pi-\frac{\pi}{4}=\frac{3\pi}{4}$

      3. $Argz$가 $-\pi,\pi$를 넘어설 수 없음을 기억할 것.

  3. 극형식의 곱셈과 나눗셈

    1. 곱셈
      1. $\left| { z }{ 1 }{ z }{ 2 } \right| =\left| { z }{ 1 } \right| \left| { z }{ 2 } \right| $
      2. $arg({ { z }{ 1 }z }{ 2 })=arg{ z }{ 1 }+arg{ z }{ 2 }$
    2. 나눗셈
      1. $\left| \frac { { z }{ 1 } }{ { z }{ 2 } } \right| =\frac { \left| { z }{ 1 } \right| }{ \left| { z }{ 2 } \right| } $
      2. $arg\frac { { z }{ 1 } }{ { z }{ 2 } } =arg{ z }{ 1 }-arg{ z }{ 2 }$
    3. 예제 3
      1. $Argz_1)$ $tan^{-1}\frac{2}{-2}=-\frac{\pi}{4}, Argz_1=\pi-\frac{\pi}{4}=\frac{3\pi}{4}$
        $Argz_2)$ $tan^{-1}\infty=\frac{\pi}{2}$
      2. $Argz_1+Argz_2=\frac{5\pi}{4}=-\frac{3\pi}{4}$
        1. $Argz$는 $\pi$를 넘으면 안 되므로 $2\pi$를 뺀다.
        2. Q. 왜 $Argz_1$에서 갑자기 $\pi$가 나오죠?
          A. 바로 위에 교수님이 강조하신 $tan^{-1} 관련 필기 참조.$
    4. 예제 4
      1. 식 : $cos^2\theta=\frac{1+cos2\theta}{2}, sin^2\theta=\frac{1-cos2\theta}{2}, sin2\theta=2sin\theta cos\theta, cos2\theta=cos^2\theta-sin^2\theta$
        1. 이 식만 알면 증명 가능.

  4. 근Roots

    1. $z=\omega^n$
      1. $z=r(cos\theta+isin\theta),
        \omega=R(cos\phi+isin\phi)$
    2. $\omega^n=R^n(cosn\phi+isinn\phi)$
      1. $R^n=r$이므로 $R=\sqrt[n]{r}$
      2. 편각 $n\phi=\theta$가 같은데,
        $\theta$가 $2\pi$의 정수배이므로
        $n\phi=\theta+2k\pi$
        $\phi=\frac{\theta+2k\pi}{n}$
    3. $\sqrt[n]{z}=\sqrt[n]{r}(cos\frac{\theta+2k\pi}{n}+isin\frac{\theta+2k\pi}{n})$
    4. 주값principal value
      1. $Argz, \sqrt [ n ]{ z }$를 주값이라 한다.
        1. argz 가 아니라 Argz 이다.
    5. 예제(Root 구하기)
      1. $z=1$
        1. $r=\left|z\right|=1, Argz=tan^{-1}0=0$
        2. $\sqrt[n]{1}=cos\frac{2k\pi}{n}+isin\frac{2k\pi}{n}$
      2. $z=1, n=3$
        1. $r=1, Argz=tan^{-1}=0$
        2. $\sqrt[3]{1}=cos\frac{2k\pi}{3}+isin\frac{2k\pi}{3}$
          1. $k=0)$ $1(cos0+isin0)=1$
          2. $k=1)$ $1(cos\frac{2\pi}{3}+isin\frac{2\pi}{3})=-\frac{1}{2}+i\frac{\sqrt{3}}{2}$
          3. $k=2)$ $1(cos\frac{4\pi}{3}+isin\frac{4\pi}{3})=-\frac{1}{2}-i\frac{\sqrt{3}}{2}$
      3. 챕터 문제
        1. 4번 : -4
          1. $r=4$
          2. $tan^{-1}\frac{0}{-4}=0$
            $Argz=\pi-0=\pi$
            1. 중요 : 좌표축에 찍어보면 $\pi$임을 확인 가능
            2. $-\pi$도 답이 될 수 있음.
        2. 13번 : $(1-i)^{20}$
          1. $1-i=\sqrt{2}(cos-\frac{\pi}{4}+isin-\frac{\pi}{4})$
          2. 이 식으로도 풀 수 있음. 답은 -1024

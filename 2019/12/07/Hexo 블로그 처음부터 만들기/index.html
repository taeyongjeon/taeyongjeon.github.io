블로그 관리

Hexo 블로그 처음부터 만들기

설치

1) PC에서 준비할 것

Git

  1. https://git-scm.com/ 에서 깃 설치

  2. 바탕화면 마우스 우클릭 - Git Bash Here 실행

    Node.js 및 Hexo

  3. https://nodejs.org/ko/ 로 이동하여 다운로드

    1. Hexo는 Node.js기반의 정적 사이트 생성 라이브러리이기 때문에
      Hexo를 사용하기 위해서는 Node.js설치가 필요하다.

  4. Hexo 설치

    1. CMD 실행
    2. Hexo 블로그 Github 연동 설정
      1. 블로그 디렉터리의 _config.yml 파일을 연 뒤 다음과 같이 수정.
      type, repo 앞에 스페이스바 2칸 띄워야 한다.

  5. 페이지 관리 디렉터리 생성

    1. 블로그 폴더를 만들 경로에서 CMD

      2) Github에서 준비할 것

  6. Github 레포지토리 생성

    1. Github 로그인
    2. New Repository로 계정이름.github.io이라는 이름의 레포지토리 생성.
      1. 예)username.github.io

        Hexo 사용법

        1) Github에 업로드

  7. 디렉터리 경로에서 CMD

    1. _config.yml 파일에서 설정한 Github.io 경로로 블로그 파일들이 업로드된다.
    2. https://계정명.github.io로 접속하여 확인할 수 있다.

      2) 로컬에서 확인하기

  8. 디렉터리 경로에서 CMD

  9. https://localhost:4000 으로 접속

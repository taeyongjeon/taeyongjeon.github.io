4학기(2019-2)►성격과적응

성격과적응 - 성격과적응 기말고사 정리

  1. 사람들
    1. 인지 Beck Ellis
    2. 인본 Maslow Rogers(ideal self)
    3. 특성 Allport, Eysenck, Cattel(Spearman), Big five
    4. 자기 Higgins(ideal self 이런거)
    5. 직업흥미검사 Holland
    6. 사랑 Murstein
    7. 욕구 William Glasser
    8. 스트레스 Seyle, McEwen
    9. 명상 Jacobson, Schultz와 Luthe, Benson
  2. 단어들
    1. REBT(합리적 정서행동치료, Rational Emotive Behavior Therapy)
    2. Beck의 인지타당성 평가 : A FROG
    3. ABC 치료모형 사건 신념 결과 논박 효과 새로운 느낌
    4. 고유자아 발달 : 신-정존확상-합고
    5. Eysenck 성격 3유형 : 내/외향성, 안정성, 신경증
    6. Cattell 성격 5요인 : OCEAN
    7. 흥미검사 RIASEC : 현실형 탐구형 예술형 사회형 기업형 관습형
    8. 자존감 높이기 : 평가 영역 바꾸기(환경 바꾸기), 하향비교, 비합리적 사고 찾기, 이중기준, 손익분석
    9. Socail Skill : 덜 진지해지기, 공감하기, 만날 때 웃기, 적절하게 감정을 표현하기
    10. 대인관계 이론들 : 주체성 연대성, 지배성 친애성
    11. 연합원리 강화원리 사회적 교환원리 형평이론
    12. 호감 : 근접성(친숙함, 유사성, 가용성) + 유사성
    13. 호감 : 구성(익숙함), 결정(근접성,유사함)
    14. 사랑 : 구성(열정), 결정(선호특성의 보유 유무, 상호 호감)
    15. Murstein의 SVR 이론 : 여과망 이론. S자극V가치R역할
    16. William Glasser의 5가지 기본욕구 이론 : 생 사 힘 자 즐
    17. 1-1-5-5-5 (여자<남자) : 매력적인 이성 / 2-5-1-1-3 (남자) : 마마보이 / 5-1-5-3-3 (남자) : 학대 / 33333 : 사랑욕구 3 남자평균이상 여자평균이하
    18. 스트레스의 대처 : 회피 도피 기술습득(상황회피) / 정서방출, 명상, 이완요법, 운동, 기분전환(반응초점적) / 엘리스 ABC 모델, 긍정적 재해석(평가중심) / 능동적 대처, 도움, 경쟁 활동 억제(문제초점적)

인지적 접근

  1. 단어 적기
    1. REBT
      1. 합리적 정서행동치료
  2. 인지타당성 평가
    1. AFROG
      Alive Feel Reality Others Good
  3. 엘리스
    1. ABC
      Activating events
      Belief
      Consequences
    2. 합리적 생각의 특징
      1. 융통성 : 빈도부사없음
      2. 현실성
      3. 기능적 유용성 : 삶을 행복하게 하는가?
    3. ABC 치료모형
      = REBT
      = ABCDEF

      인본주의적 접근

  4. 매슬로우 20세 결혼
    1. 욕구
      1. 생리적(유년기)
        안전 욕구(유년기)
        사랑과 소속의 욕구(청년기)
        존중 욕구(청년기)
        자아실현 욕구(인생의 중반 이후)
  5. Rogers 현상학
    1. 자아(self) : real, ideal
    2. 양육 : 무조건적 긍정적 수용적. 가치의 조건화(if then) 극혐
    3. 내담자 중심치료(비지시적 치료)
      1. 일치성, 진솔성, 무조건적 긍정적 존중, 적극적 경청, 공감적 이해

        특성적 접근

  6. Allport
    1. 성격(역동조직), 특성(유일접근방식), 개인특성(주-중심-이차)
    2. 습관이 모여 특성, 태도(참조대상있음), 자아(통합하는 힘), 고유자아(개인성격 합)
    3. 고유자아 발달
      1. 신체적 자아
        자아정체감
        자아존중감
        자아확장
        자아상
        합리적 적응체제로서 자아
        고유자아 추구
  7. Eysenck 히포크라테스 제자
    1. 성격 3유형 : 내/외향성, 안정성, 신경증
    2. 우울질(우울함), 담즙질(성질), 점액질(냉담), 다혈질(쾌할한)
    3. 성격은 유전에서 옴. 조건형성, 이상행동
  8. Cattell
    1. 스피어맨
    2. 특성, 표면특성, 근원특성
    3. 16PF(16 Personality Factors)
    4. 16PF 대체용으로 5요인 만듬. NEO PI R이라 부름.
      성격 5요인 OCEAN
      Openness 개방성
      Conscientiousness 성실성
      Extraversion 외향성
      Agreeableness 우호성
      Neuroticism 신경증

적성과 흥미

  1. 직업적성검사
    1. 80, 120 : 10%
  2. 직업흥미검사
    1. Holland
    2. 개인의 선호 검사임. 다른사람과 비교 X
    3. 흥미검사+5요인성격검사+생활사
  3. 흥미검사 RIASEC
    1. 40/60 뚜렷
    2. 응답신뢰성 : 사회적 바람직성, 부주의성
  4. 생활사검사
    1. 40/60 뚜렷

직업가치관검사

  1. 반구조화면접
  2. 몸과 마음의 여유, 자율, 인정, 애국 등 13요인

자기, 자존감

  1. 자기 : 물질적, 정신적, 사회적 자기
  2. 자기개념 : 신체로, 감각으로(18개월), 자기인지로(2-5세), 언어로(3-5), 심리적 특성적으로(초등학생 이후) 발달
  3. 반영평가, 비교 : 공적 자기와 사적 자기
  4. 자기파악 : 자기지각(내성, 행동관찰), 비교평가, 반영평가
  5. Higgins : 실제, 이상(낙담정서), 당위(불안정서)
  6. 자존감 : 일상과제(2세), 안정애착(애정,통제), 사회비교(지적 사회적 신체적 측면), 안정적(초3), 감소(사춘기)
  7. 자존감 높이기 : 평가 영역 바꾸기(환경 바꾸기), 하향비교, 비합리적 사고 찾기, 이중기준, 손익분석

대인관계

  1. 가족 낭만적 직업적 사회적 동반자
  2. 대상관계(양육자와 형성한 관계)이론 : 프로이트, 영국 정신분석학회
  3. 인간의 두 동기 : 주체성 연대성, 대인관계에서 지배성, 친애성으로 이어짐
  4. 자기상 : 주체적 자기상, 관계적 자기상
  5. 호감 : 근접성(친숙함, 유사성, 가용성) + 유사성
  6. Socail Skill : 덜 진지해지기, 공감하기, 만날 때 웃기, 적절하게 감정을 표현하기
  7. 대인관계 이론들
    1. 연합원리 : 행동주의. 친해지려면, 무조건 자극(이벤트)을 제공해라.
    2. 강화원리 : 칭찬칭찬칭찬선물선물선물
    3. 사회적 교환원리 : 보상-비용 = quality. 높으면 만족, 안정성.
    4. 형평이론 : 아냐, 안정성은 ‘대안 존재여부’를 고려해야 돼. 결과/투입 OC 결과/투입
  8. 치명적 의사소통 : 비난 모욕 자기변호 도피

이성관계

  1. 호감 : 구성(익숙함), 결정(근접성,유사함)
  2. 사랑 : 구성(열정), 결정(선호특성의 보유 유무, 상호 호감)
  3. 사랑 못하는 경우 : 안정애착X, 자율성(어린시절 규제로 형성), 자존감, 완벽추구, 거부불안
  4. 유형 : 논리/낭만/소유/이타/유희
  5. 사랑의 삼각형이론 : 열정 친밀감 의사결정 및 헌신으로 이뤄짐.
    1. 친밀감(호감), 열정(풋사랑), 헌신(애정 없는 부부)
    2. 친밀감+열정(낭만적), 친밀감+헌신(부부에), 열정+헌신(얼빠진 사랑)
    3. 친밀감+열정+헌신(완전한 사랑)
  6. Murstein의 SVR 이론 : 여과망 이론. S자극V가치R역할

욕구

  1. 욕구 : 필수적 물질의 결핍상태. drive가 생겨남. 결핍 보상deprived reward으로 해소-도파민 관련
  2. 생리적, 심리적 욕구, 심리적 욕구 : 심적자원(지배 인정 욕구 등) 부족시 발생.
  3. 욕망 : 충족되지 않은 목표가 있을 때 발생. motivation이 생겨남. 비결핍 보상nondeprived reward으로 해소
  4. William Glasser의 5가지 기본욕구 이론 : 생 사 힘 자 즐
  5. 1-1-5-5-5 (여자<남자) : 매력적인 이성 / 2-5-1-1-3 (남자) : 마마보이 / 5-1-5-3-3 (남자) : 학대 / 33333 : 사랑욕구 3 남자평균이상 여자평균이하

스트레스와 질병, 대처

  1. Seyle : 변화요구사건이 스트레스
  2. 생리적 반응 : 교감신경, ACTH(에너지증가) / 정서반응 : 짜증, 분노 / 행동반응 : 대처
  3. Seyle 적응증후군 : 경고저항소진
  4. 스트레스의 대처 : 회피 도피 기술습득(상황회피) / 정서방출, 명상, 이완요법, 운동, 기분전환(반응초점적) / 엘리스 ABC 모델, 긍정적 재해석(평가중심) / 능동적 대처, 도움, 경쟁 활동 억제(문제초점적)

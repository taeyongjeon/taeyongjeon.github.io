리눅스 우분투 Git 사용법

설치

설정

만약 정보를 잘못 입력했다면 다음 명령어로 일괄 삭제할 수 있다.

원격 -> 로컬 최초 다운로드 : clone

mkdir 명령어로 폴더를 만들 필요가 없다.
주소는 각 리포지토리에서 Clone or download를 누르면 복사할 수 있다.

원격 <-> 로컬 일상작업 : pull / commit / push

여러 사람들이 함께 개발한다면 git pull로 변경사항이 없는 지 업데이트한다.

코드를 수정했다면 원격 저장소에 add, commit, push로 올릴 수 있다.

기타 : SSH Key 인증 : pull, push 때 username과 비밀번호 안 물어보게 하기

  1. 먼저 공개키를 생성한다.
  2. Settings - SSH and GPG Keys로 이동한다.
  3. Title은 아무거나 입력, keys는 방금 복사한 내용을 붙여넣기한 뒤 Add SSH Key 클릭.
  4. 설정은 모두 끝났고 다음 작업을 수행하면 된다.
    1. 아직 clone되지 않은 리포지토리
      1. Clone or download를 누르고 Use SSH를 눌러 나오는 링크를 복사한다.
      2. 이걸로 git clone git@github.com:facebook/react-native.git
      3. 이제 pull, push 등이 별도 인증이 필요없어진다.
    2. 이미 clone된 리포지토리
      1. Clone or download를 누르고 Use SSH를 눌러 나오는 링크를 복사한다.
      2. 해당 폴더 이동 후 git remote set-url origin git@github.com:facebook/react-native.git
      3. 이제 pull, push 등이 별도 인증이 필요없어진다.
